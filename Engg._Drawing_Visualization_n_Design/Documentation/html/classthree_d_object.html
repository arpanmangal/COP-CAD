◆ rotate()

void threeDObject::rotate ( int  rotMatrix[][3])

The Rotate function takes a rotation matrix as input as a 3*3 array It applies the rotation matrix to each and every point of the point Set. Since points are pointers to actual points, it will modify actual points, in turn automatically modifying the edges and the current object. It's the job of the controller to provide the rotation matrix

Given a 3*3 rotation matrix M, a Point P (x, y, z) will translate to M*P We iterate through each point in the pointSet to do the transformation. Edge transformation follows automatically
